## Riscos identificados

- Vazamento de CID via XML mal-formado interceptado em transit
- Glosa retorna com PII de outros pacientes (caso operadora aplique CRC errado)
- Reuso de XML em outra operadora sem consent renovado

## Medidas de mitigação

- TLS 1.3 em transit; mTLS quando operadora suportar
- Validador XSD + regras de negócio antes de transmitir (ADR 0079 Fase 2)
- Hash chain do XML emitido (regra 39) — irretratabilidade
- Segregação por operadora em `tenant.tiss_credentials`
