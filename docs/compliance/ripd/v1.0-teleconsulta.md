## Notas críticas (a expandir)

- **CFM 2.314/2022 telemedicina** — gravação só com consent expresso; profissional médico responsável; emissão de prontuário ao final
- Provider de vídeo (Daily.co, LiveKit, Twilio Video — a definir) — sub-processor LGPD declarado em [DPO](../dpo.md)
- Retenção da gravação: **opcional** (consent específico) — se não, descartada após geração do SOAP; se sim, mesma retenção do prontuário
- Transcrição via Groq Whisper (sub-processor — declarar) — texto pode conter dados de outras pessoas mencionados; sanitizar
- SOAP automático classe II → exige Comitê IA + supervisão humana (regra 28) + classificador output (regra 28) + `ai_audit_log`
- Cross-tenant: teleconsulta nunca cruza tenant — só resumo via passaporte (regra 42)
- Threat model dedicado em [`docs/threat-models/`](../../threat-models/) — listar quando criado (intervenção em stream, MITM, vazamento de gravação)
