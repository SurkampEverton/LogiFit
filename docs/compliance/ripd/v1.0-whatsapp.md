## 1. Identificação do tratamento

- **Nome do módulo:** WhatsApp Business bidirecional — paciente envia mensagem ao tenant ou recebe template (cobrança/lembrete); sistema registra conversa, anexos (foto, áudio, PDF) e roteia para profissional ou bot via régua declarativa + intent router IA
- **Sprint que entrega:** [Sprint 13 — WhatsApp + Régua de Cobrança](../../sprints/13-geral-whatsapp-e-regua-cobranca.md)
- **ADR de referência:** ADR 0025 (esperado Sprint 13) — Provider WhatsApp (Twilio / Z-API / Meta direto) + ADR 0026 (esperado Sprint 13) — Motor declarativo de régua DSL JSON
- **Marco regulatório:**
  - **LGPD art. 7 V** (execução de contrato) + **art. 11 II "a"** (quando dado de saúde por mensagem)
  - **Termos do WhatsApp Business** (Meta) — DPA aplicável
  - **Resolução ANPD nº 18/2024** (canal de atendimento via mensageria)
- **Finalidade declarada:** receber mensagens de pacientes/leads para atendimento, agendamento e lembrete; paciente envia spontaneamente ou em resposta a template
- **Categorias de dado tratado:**
  - [x] Identificação (telefone, nome do contato no WhatsApp)
  - [x] Saúde (quando paciente menciona sintoma/condição)
  - [x] Mídia (foto de exame, áudio, PDF)
  - [ ] Biometria
  - [ ] Genética

## 2. Base legal

| Categoria | Base legal | Justificativa |
|---|---|---|
| Mensagens texto | art. 7 V (contrato) + art. 11 II "a" se saúde | Paciente inicia contato; consent presumido na adesão; opt-out em `STOP` |
| Mídia (exame, áudio descrevendo sintoma) | art. 11 II "a" | Mesmo enquadramento; mídia segue regra 38 (scanUpload) |
| Conversas com bot/IA | art. 7 V + classificador anti-prescrição (regra 28) | Bot **não** dá diagnóstico; gate redireciona ao profissional |

## 3. Princípios (skeleton)

- **Finalidade:** atendimento ao paciente do tenant; **não** alimenta IA cross-tenant; **não** é vendido
- **Adequação:** apenas conversas vinculadas a member/lead conhecido do tenant
- **Necessidade:** retenção 2 anos pós-última mensagem; mídia 1 ano
- **Segurança:** webhook HMAC + scanUpload + classificador anti-prescrição em qualquer resposta IA + audit
- **Transparência:** template de boas-vindas explica privacidade + opt-out

## 4. Fluxo de dados (a expandir Sprint 18)

```
Paciente envia mensagem → WhatsApp Business API webhook (Meta) →
  Server Action wrapAction (HMAC validation + idempotência) →
    INSERT messages (RLS por tenant + scope conversa) →
      se mídia: scanUpload (regra 38) → Cloudflare R2 cifrado →
        roteamento: bot (com gate anti-prescrição) OU profissional
```

## 5. Medidas de segurança

| Camada | Controle | Referência |
|---|---|---|
| Webhook validation | HMAC + idempotência via `external_id` | regra 33 |
| Rate limit | por tenant + por número origem | regra 36 |
| Upload scanning | `scanUpload()` para mídia | regra 38 |
| IA classifier | anti-prescrição obrigatório em respostas IA | regra 28 |
| Isolamento | RLS multi-tenant; conversas nunca cruzam tenant | regra 1 |
| Retenção | mensagens 2a; mídia 1a; opt-out paciente apaga histórico em `/meu/privacidade` | regra 34 |
| Cross-tenant | conversa nunca cruza tenant | regra 42 |

## 6. Avaliação de risco

| Risco | Probabilidade | Impacto | Mitigação |
|---|---|---|---|
| Paciente compartilha exame em foto sem proteção | alta | médio | scanUpload + criptografia + audit + UX educa o paciente |
| Bot dá conselho clínico não-supervisionado | média | crítico | Classificador anti-prescrição + gate `withAiClassGate` em respostas IA + escalada obrigatória ao profissional para tópicos clínicos |
| Vazamento via Meta (sub-processador) | baixa | alto | DPA Meta + criptografia E2E padrão WhatsApp + LogiFit não retém token Meta cifrado em log |
| Spam de leads não-membros | média | baixo | Rate limit por número + bloqueio manual + lista de números bloqueados |
| Tenant cruza canal: lê conversa de outro tenant | baixa | crítico | RLS + audit + lint multi-tenant em queries (regras 1-2) |

## 7. Direitos do titular

- Paciente envia `STOP` → bot bloqueia novas mensagens (mas histórico retido por 2a até `/meu/privacidade` apagar)
- Histórico em `/meu/privacidade/conversas` (Sprint 26)
- Solicitar exclusão completa em `/meu/privacidade` → cessa retenção + delete (preservando logs operacionais anonimizados)
- SLA **15 dias úteis**
