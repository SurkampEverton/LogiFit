## 1. Identificação do tratamento

- **Nome do módulo:** Pipeline de exames laboratoriais — upload PDF/imagem → OCR → extração de analitos → interpretação conservadora com IA → revisão humana obrigatória
- **Sprint que entrega:** [Sprint 33 — Pipeline exames laboratoriais](../../sprints/33-geral-pipeline-exames.md)
- **ADR de referência:** [ADR 0050 — Pipeline exames laboratoriais](../../decisions/0050-pipeline-exames-laboratoriais.md)
- **Marco regulatório:**
  - **CFM 2.454/2026** (IA em medicina — supervisão humana obrigatória, Comitê IA do tenant)
  - **ANVISA RDC 657/2022** (SaMD Classe II — notificação simples obrigatória)
  - **LGPD art. 11 II "a"** (tutela da saúde)
- **Classificação SaMD:** Classe II ([samd-classification.md](../samd-classification.md))
- **Finalidade declarada:** apoiar profissional na interpretação de exames com IA conservadora; **decisão clínica permanece humana**
- **Categorias de dado tratado:**
  - [x] Saúde (resultados laboratoriais, valores numéricos, faixas de referência)
  - [x] Identificação (paciente, requisitante, laboratório de origem)
  - [ ] Genética (não no MVP — exames genéticos têm RIPD próprio futuro)
  - [ ] Biometria

## 2. Base legal

| Categoria | Base legal | Justificativa |
|---|---|---|
| Resultados laboratoriais | art. 11 II "a" — **tutela da saúde** + consent específico de IA (regra 28) | Paciente consente uso de IA na interpretação ao aceitar termos do tenant + opt-out via `/meu/privacidade` |
| Documentos originais (PDF) | art. 11 II "a" + retenção legal do laudo (5 anos no MVP — alinha a obrigação fiscal de exames quando lançados em conta) | Documento original armazenado cifrado (Cloudflare R2 → S3 fase 2) |

## 3. Princípios aplicados (skeleton)

- **Adequação:** somente exames cuja interpretação cabe ao escopo profissional do tenant (CFM/CFN/COFFITO conforme caso) entram no pipeline
- **Necessidade:** OCR extrai apenas analitos relevantes; campos livres do laudo não são indexados em vetorial sem opt-in
- **Segurança:** RLS multi-tenant + criptografia at-rest + classificador anti-prescrição (regra 28) + supervisão humana obrigatória
- **Não discriminação:** modelo IA conservador (recusa interpretar quando confiança baixa) + auditoria de viés trimestral

## 4. Fluxo de dados (a expandir Sprint 33)

```
Upload PDF/JPG → scanUpload (regra 38) → OCR Vertex AI (BR) →
  extração analitos (regex+LLM com schema Zod) →
    classificador SaMD: gate Comitê IA (regra 28) →
      interpretação conservadora → ai_audit_log (input/output/modelo/decisão humana) →
        revisão obrigatória profissional → liberação ao paciente
```

## 5. Medidas de segurança

| Camada | Controle | Referência |
|---|---|---|
| Upload scanning | `scanUpload()` MIME real + magic bytes + ClamAV | regra 38 |
| IA gate | `withAiClassGate(featureKey)` + Comitê IA + ata | regra 28 + ADR 0053 |
| Auditoria IA | `ai_audit_log` (input, output, modelo, decisão humana, samd_class) | regra 28 |
| RAG isolado | embeddings somente do próprio tenant — nunca cross-tenant | ADR 0064 |
| Cota IA | hard-stop por plano + BYOK opcional | ADR 0064 + 0066 |
| Retenção | PDF original 5a; analitos estruturados 20a junto a prontuário | regra 34 + ADR 0072 |

## 6. Avaliação de risco

| Risco | Probabilidade | Impacto | Mitigação |
|---|---|---|---|
| IA dá interpretação errada e profissional não revisa | baixa | crítico | Supervisão humana **bloqueante** (paciente só vê após profissional liberar) + classificador anti-prescrição |
| Vazamento de PDF de exame | baixa | alto | RLS + criptografia + audit + signed URL Cloudflare R2 |
| Bias algorítmico em determinados perfis (idade, sexo, raça) | média | médio | Auditoria interna trimestral + opção de fallback "interpretação manual" |
| Tenant ativa feature sem Comitê IA | baixa | crítico | Gate `withAiClassGate` bloqueia hard — feature flag não passa |

## 7. Direitos do titular

- Paciente vê em `/meu/exames` o resultado + indicação de que IA participou da interpretação + nome do profissional revisor
- Direito de revisão de decisão automatizada (LGPD art. 20) — `/meu/alertas-ia` (Sprint 26) permite contestar
- SLA **15 dias úteis**
